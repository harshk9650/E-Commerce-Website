# E-Commerce Website 🛒

🚀 **Live Demo:** [E-commerce Website](https://e-commercewebsiteeeee.netlify.app/)  

A modern **multi-vendor e-commerce platform** built with **Next.js** and **Tailwind CSS**. This application allows multiple vendors to manage their own stores while customers enjoy a seamless online shopping experience.

## ✨ Features

### 🏪 Multi-Vendor Architecture
- Vendors can register and create their own stores
- Manage products, pricing, inventory, and orders
- Separate dashboards for each vendor

### 🛍️ Customer Storefront
- Clean and responsive shopping UI
- Browse products by category
- Product details page with pricing and images
- Add to cart and checkout flow

### 📊 Vendor Dashboard
- Add, update, and delete products
- View sales reports and order analytics
- Manage customer orders efficiently

### 🛠️ Admin Panel
- Manage vendors and users
- Approve or remove products
- Track commissions and total platform sales
- Full control over marketplace operations

### 🎨 User Experience
- Fully responsive design
- Modern UI with Tailwind CSS
- Fast navigation and smooth interactions
